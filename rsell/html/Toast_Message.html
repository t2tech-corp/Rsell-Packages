Toast_Message {rsell}R Documentation

Generate Toast Message

Description

This function generates a toast message.

Usage

Toast_Message(tr_message, tr_type)

Arguments

tr_message

The message for the Toast output.

tr_type

The type of Toast message to generate.

Details

Requires: shinytoastr

Value

A Toast message at the bottom right of the page.
