Secure_Check {rsell}R Documentation

Perform Security Check on Tab Access

Description

This function performs a security check for the tab to be accessed.

Usage

Secure_Check(input_tab, tab_name, user_auth)

Arguments

input_tab

Current Tab

tab_name

Target Tab Name

user_auth

Authorized User

Details

Requires:

Value

auth_valid Logical return to allow/deny access.
