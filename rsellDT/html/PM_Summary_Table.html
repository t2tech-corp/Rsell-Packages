PM_Summary_Table {rsellDT}R Documentation

Creates the Poshmark Summary Data Table

Description

This function creates a Poshmark Summary Data Table.

Usage

PM_Summary_Table(summary_table, table_type)

Arguments

summary_table

The table to be used for the report.

table_type

The table type for the report.

Details

Requires: DT

Value

A Poshmark Summary Data Table.
