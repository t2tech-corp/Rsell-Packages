Inventory_Table {rsellDT}R Documentation

Creates the Inventory Detail Data Table

Description

This function creates a Inventory Detail Data Table.

Usage

Inventory_Table(inv_table, rem_flag = FALSE)

Arguments

inv_table

The inventory table to generate the report.

rem_flag

The flag to indicate whether the Edit Button should be removed from output.

Details

Requires: DT, data.table

Value

An Inventory Detail Data Table.
