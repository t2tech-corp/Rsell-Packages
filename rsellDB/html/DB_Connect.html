DB_Connect {rsellDB}R Documentation

Connect to PostgreSQL Database

Description

This function connects to the PostgreSQL Database.

Usage

DB_Connect(db_name, db_host, db_user, db_pass)

Arguments

db_name

The name of the database.

db_host

The database hostname.

db_user

The database user ID.

db_pass

The database password for the user ID.

Details

Requires: RPostgres

Value

db_con The connection parameters to the database.
